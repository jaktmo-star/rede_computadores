## 📚 Sobre o Projeto
Este projeto simula uma rede corporativa contendo os departamentos:
- Engenharia
- Compras
- TI Interno
- Infraestrutura

Foram utilizados conceitos de:
- VLAN
- DHCP
- Endereçamento IP
- Sub-redes
- Topologia estrela
- Configuração de switches Cisco

## 🛠 Ferramentas Utilizadas
- Cisco Packet Tracer
